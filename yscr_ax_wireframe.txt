=====================================================
UPDATED WIREFRAME — ADMIN & FAN VOTING INDEPENDENT
(WITH FAN-ONLY SUDDEN DEATH)
=====================================================

CORE RULE (VISIBLE IN SYSTEM, NOT UI)
------------------------------------
- Fans and Admins vote independently
- No weighting, no merging, no override
- Same matches, separate outcomes
- Fan ties are resolved ONLY by fans (Sudden Death)
- Admins never resolve fan ties

SCREEN 2A: Sudden Death Voting (Fan – Tie Only)
----------------------------------------------------
|              ⚠ SUDDEN DEATH ⚠                  |
| Event: Dakar Rally | Class 450 | Heat 1         |
| Match: 3 of 12                                 |
| Timer: 00:15                                   |
|--------------------------------------------------|
|                     HOTSEAT                     |
| Rider A → Rider B                               |
| Description: Pass on turn 3                     |
|--------------------------------------------------|
|                   CHALLENGER                    |
| Rider C → Rider D                               |
| Description: Pass on final straight             |
|--------------------------------------------------|
|        [ Vote HOTSEAT ]  [ Vote CHALLENGER ]     |
----------------------------------------------------
Notes:
- Triggered ONLY if Screen 2 ends in a tie
- Same sessions that voted in the original match are eligible
- Maximum of 3 Sudden Death rounds
- Timer per round: 15 seconds
- One additional vote per session per round
- No new voters allowed
- Timeout → vote not counted (no default winner)
- Resolution:
   1. First side to reach strict majority (>50%) wins
   2. If no majority after 3 rounds → match declared FAN TIE
- Admins are NOT notified and cannot intervene
